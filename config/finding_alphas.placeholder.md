# Finding Alphas Placeholder

Expected file:
- `knowledge_base/books/finding_alphas_2ed.pdf`

Why this is a placeholder:
- `Finding Alphas` is a copyrighted book.
- This repository does not auto-download copyrighted books from unofficial sources.

How to enable it:
1. Obtain a legal copy of the PDF.
2. Copy it to `knowledge_base/books/finding_alphas_2ed.pdf`
3. Or set `FINDING_ALPHAS_PDF_PATH` and run `scripts/import_finding_alphas.ps1`

Once present, the knowledge loader will index it automatically because PDF support is enabled.
